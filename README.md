# ls-for-win
The ls command from unix to windows

# Setup / Installation
Make a folder in "C:" named bin, and move the ls.exe to the path "C:\bin". Then open "edit the system environment variables" from the windows start menu. click on environment variables then Path from the above section. Click on edit<new then paste "C:\bin".

# Usage
Just like Linux and unix based os, open a terminal or navigate from the terminal to any path or folder, then type ls to list all the files and folders.

# WARNING:
It is yet very primitive and does not suport any arguments like ls -a, it may come in later versions so make note, this is not the full ls tool yet.
